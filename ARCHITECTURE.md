# Nexora AI Architecture

## High-Level Architecture

```text
Frontend (React)
        ↓
Backend API (FastAPI)
        ↓
AI Engine (Python + NLP)
        ↓
Database (PostgreSQL)
```

---

## AI Workflow

```text
Resume Upload
      ↓
Resume Parsing
      ↓
Skill Extraction
      ↓
Industry Skill Mapping
      ↓
Skill Gap Analysis
      ↓
Job Readiness Score
      ↓
Recommendations
```
