# Nexora AI

AI-powered employability intelligence platform for skill gap analysis, placement prediction, and talent analytics.

## Features

- Resume Parsing
- Skill Gap Analysis
- Job Readiness Score
- AI-Based Recommendations
- Placement Prediction
- Recruiter Matching
- College Analytics Dashboard

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- FastAPI

### AI/ML
- Python
- spaCy
- scikit-learn
